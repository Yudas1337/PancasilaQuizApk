# PancasilaQuizApk

 A Simple Quiz Game Using Kotlin
 
 This Repo is for the Apk to play the game and the data is from the rest server from quizPancasila Repo's


# Crafted With :
<ul>
<li>Retrofit </li>
<li>Facebook Shimmer</li>
<li>Picasso</li>
<li>CircleImageView</li>
<li>CountdownTimer</li>
</ul>
